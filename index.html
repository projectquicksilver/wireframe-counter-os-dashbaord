Retail Intelligence — Recommended tabs & sections
Four tabs covering the full retailer lifecycle — from discovery and health to competitive positioning.
01
Retailer Overview
Top-level pulse on the entire retailer base
Summary metrics strip header
Total retailers — Active vs. inactive breakdown
New onboarded this month vs. last month
Average purchase value per retailer
Churned retailers (no purchase in 60 days)
Retailer health map map
Geographic view — dots colored by activity tier (green / amber / red)
Toggle: sales volume vs. verification rate vs. last purchase date
Click-to-expand retailer card with contact, GPS, and history
Activity distribution (Bar chart) chart
Purchases per month — histogram by retailer bucket (0-1, 2-5, 6-10, 10+)
SKU breadth per retailer — how many product lines each stocked
02
Retailer Segments
Tiered view to prioritize field effort
Segment cards (Star / Gold / Regular / At-risk / Churned) tier
Count of retailers per tier — with revenue contribution %
Tier migration: how many moved up or down this quarter
Retailer list (filterable table) table
Filter by tier, territory, purchase recency, SKU, field agent
Columns: Retailer name, tier badge, last purchase, lifetime value, verified %, field agent
Actions: Assign agent, add note, flag for follow-up, view profile
Retailer profile drill-down detail panel
Purchase history — timeline and bar chart
SKU affinity — which products this retailer consistently stocks
Verification rate trend — monthly line chart
Field agent visit log with notes
Next recommended action — AI generated (e.g. "upsell NPK based on Kharif season")
03
SKU Intelligence
Which products are moving at retail level — and which aren't
SKU performance table table
Retailer penetration per SKU — % of retailers who stocked it
Sell-through rate — units sold vs. units ordered
Seasonal velocity — compare Kharif vs. Rabi cycle
SKU cross-sell matrix heatmap
Which SKU pairs are most often bought together at retail level
Highlight gaps — e.g., retailers who buy Nano Urea but not Zyme
Actionable: auto-suggest bundle to field agent before visit
Dead stock alerts alert
Retailers with ordered SKU not yet scanned / verified after N days
Send in-app nudge to field agent for a follow-up visit
04
Geo Coverage Intelligence
Spotting white-space — where retail presence is missing
Uncovered village list gap
Villages with no active retailer within N km — ranked by population / crop potential
One-click: assign nearest field agent to prospect the area
Retailer density map map
Heat overlay showing retailer count per block / mandal
Toggle between total count and active-only count
Sales & Rewards — Recommended tabs & sections
Three layers — sales performance analytics, incentive program management, and a loyalty program for retailers and field agents.
01
Sales Performance
Deep analytics on what was sold, by whom, where, and whether it was verified
Period comparisons analytics
MTD / QTD / YTD toggle — actual vs. target vs. last year
Breakdown: verified sales vs. non-verified — waterfall chart
Top-down: National → State → Territory → Agent
Leaderboard — field agent ranking rank
Rank by: verified sales, village coverage, streak, or scan count
Badge-tier display next to each agent — Bronze / Silver / Gold / Star
Weekly movement: who climbed or dropped
Target vs. actuals table table
Each row = one agent / territory with target, achieved, gap, and % progress bar
Color code: green ≥ 100%, amber 70–99%, red < 70%
Export as CSV or PDF for review meetings
Verified sales ROI bridge chart
Waterfall bar: base cost → after verification → after loyalty → after repeat purchase → total ROI
Editable inputs — what-if simulator (change verification rate → see ROI change)
02
Incentive Programs
Create, manage and track sales contests and bonus schemes
Active programs list list
Program name, type (individual / territory / state), period, prize pool
Status: Live / Upcoming / Ended — with days remaining countdown
Progress bar — % of participants who have hit threshold
Program builder create
Step 1 — Define: name, type, eligible roles, geography scope
Step 2 — Rules: KPI (verified sales, scan count, new retailer), target, duration
Step 3 — Rewards: cash / gift / badge / extra leave / co-op store credit
Step 4 — Approval: send for HQ or state head sign-off
Program analytics analytics
Real-time participant progress — ranked list with bar chart
ROI of program: incremental sales generated vs. prize pool cost
Payout tracker — who has been paid, who is pending
03
Loyalty & Points
Retailer loyalty program — turning repeat buyers into brand advocates
Points dashboard (HQ view) overview
Total points issued this month vs. last month
Redemption rate — % of earned points that have been redeemed
Top 10 retailers by lifetime points — with tier badge
Burn liability — financial exposure of unredeemed points
Points wallet (Field Agent view) field
My points balance + streak bonus + pending approval points
Tier progress bar — e.g. "450 more points to reach Gold"
Earn history — per sale, per scan, per verified transaction
Redemption catalog — redeem for gift cards, gadgets, or co-op credit
Retailer loyalty card retailer
QR-based — retailer scans at point of purchase to earn points automatically
Tier badges shown to retailer: Bronze → Silver → Gold → Star Partner
Birthday / anniversary bonus — automated point boost events
Referral program — earn points for introducing a new active retailer
Redemption management admin
Approve or reject redemption requests with audit trail
Catalog manager — add / edit / retire reward items with point cost
Budget controls — cap on monthly redemption payout per region
